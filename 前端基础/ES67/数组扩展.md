# ES6 数组扩展详解（扩展运算符、空位、Array.from、Array.of、fill）
## 一、扩展运算符 `...`（ES6）
### 1. 基础定义
`...` 三个点，把**可迭代对象**（数组、字符串、Set、Map、arguments）展开成逗号分隔的序列。
### 2. 常用场景
#### （1）复制数组（浅拷贝）
```js
const arr = [1,2,3];
// 浅拷贝新数组，不影响原数组
const copy = [...arr];
console.log(copy); // [1,2,3]
```
对比：`arr.slice()` 同样浅拷贝，扩展运算符更简洁。

#### （2）合并数组
```js
const a = [1,2];
const b = [3,4];
const merge = [...a, ...b, 5];
console.log(merge); // [1,2,3,4,5]
```

#### （3）解构赋值配合使用（收集剩余元素）
```js
const [first, ...rest] = [10,20,30,40];
console.log(first); // 10
console.log(rest); // [20,30,40]
```
> 注意：扩展运算符只能放在解构最后，`[...rest, last]` 报错。

#### （4）函数传参，替代 arguments
```js
function sum(a,b,c) { return a+b+c; }
const nums = [1,2,3];
sum(...nums); // 等价 sum(1,2,3)
```

#### （5）字符串转数组
```js
[...'es6'] // ['e','s','6']
```

### 3. 局限性
仅**浅拷贝**，数组嵌套对象时，拷贝的是引用：
```js
const arr = [{x:1}];
const copy = [...arr];
copy[0].x = 99;
console.log(arr[0].x); // 99 原数组被修改
```

## 二、数组的空位（ES6 统一修正空位行为）
### 1. 什么是数组空位
数组逗号之间无值，称为**空位**，不等于 `undefined`
```js
const arr = [1,,3]; // 下标1是空位，不是undefined
```

### 2. ES5 空位行为（混乱、坑多）
- `forEach` / `filter` / `every` / `some`：直接跳过空位
- `map`：跳过空位，但保留空位位置
- `join`、`toString`：空位视为空字符串
```js
[1,,3].map(x=>x*2); // [2, empty, 6]
```

### 3. ES6 统一规则：空位全部转为 `undefined`
1. 扩展运算符 `...`：空位替换为 `undefined`
```js
[...[1,,3]] // [1, undefined, 3]
```
2. `Array.from()`：空位转为 `undefined`
3. `find` / `findIndex` / `fill` / `for...of` 全部访问空位，当作 `undefined`
4. `entries()`、`keys()`、`values()` 遍历空位，不跳过

### 4. 补充：创建无空位数组
`Array(3)` 生成 `[empty ×3]` 全空位；
`Array.of(3)` 生成 `[3]`，不会产生空位。

## 三、Array.from()（ES6）
### 1. 作用
将**类数组对象 / 可迭代对象**转为真正数组。
两类可转换目标：
1. 类数组：有 `length` 属性、数字下标（DOM NodeList、arguments）
2. 可迭代对象：数组、字符串、Set、Map

### 2. 语法
```js
Array.from(arrayLike, mapFn, thisArg)
```
- `arrayLike`：必传，类数组/可迭代对象
- `mapFn`：可选，生成数组时同步遍历映射（等价 `map`）
- `thisArg`：可选，map 函数内 this 指向

### 3. 基础示例
#### （1）DOM 节点转数组
```js
const divs = document.querySelectorAll('div');
const arr = Array.from(divs); // 可使用数组方法 filter/map
```

#### （2）字符串、Set 转数组
```js
Array.from('abc'); // ['a','b','c']
Array.from(new Set([1,2,2,3])); // [1,2,3] 快速去重
```

#### （3）处理数组空位（转为 undefined）
```js
Array.from([1,,3]); // [1, undefined, 3]
```

#### （4）映射参数，快速生成序列
```js
// 生成 0~9 数组
Array.from({length:10}, (_,i)=>i);
// 等价 [0,1,2,...9]
```

### 4. 对比扩展运算符 `...`
- `...` 只能转**可迭代对象**，不能转普通类数组（如 `{0:'a',length:1}`）
- `Array.from` 兼容类数组 + 可迭代，自带 map 回调，功能更强

## 四、Array.of()（ES6）
### 1. 解决原生 Array 构造函数的缺陷
原生 `Array()` 传参逻辑诡异：
```js
Array(3); // [empty ×3] 创建长度3的空位数组
Array(1,2,3); // [1,2,3] 多参数才生成数组
```
`Array.of` 统一行为：**所有参数作为数组元素，不存在创建空位数组**

### 2. 语法 & 示例
```js
Array.of(...items)
```
```js
Array.of(3); // [3] 单个数字不会生成长度空位
Array.of(1,2,3); // [1,2,3]
Array.of(); // [] 无参返回空数组
Array.of(undefined,null); // [undefined, null]
```

### 3. 简单 polyfill 理解原理
```js
if (!Array.of) {
  Array.of = function() {
    return Array.prototype.slice.call(arguments);
  }
}
```

## 五、数组实例方法 fill()（ES6）
### 1. 作用
用指定值**批量填充数组元素**，修改原数组。
### 2. 语法
```js
arr.fill(value, startIndex, endIndex)
```
- `value`：填充的值（必填）
- `startIndex`：起始下标，默认 0
- `endIndex`：结束下标（不包含），默认数组长度

### 3. 基础示例
```js
// 1. 全部填充
const arr = [1,2,3];
arr.fill(0);
console.log(arr); // [0,0,0]

// 2. 指定区间填充
[1,2,3,4].fill(9,1,3); // [1,9,9,4] 下标1、2被填充

// 3. 负数下标（倒数索引）
[1,2,3].fill(6, -2); // [1,6,6] 从倒数第二位开始
```

### 4. 高频使用场景：初始化定长数组
```js
// 创建长度5，全部填充0的数组
const list = new Array(5).fill(0); // [0,0,0,0,0]
```

### 5. 重要坑：填充引用类型
`fill` 填充对象时，所有位置指向**同一个引用**，修改一处全部改变：
```js
const arr = new Array(2).fill({num:1});
arr[0].num = 99;
console.log(arr); // [{num:99}, {num:99}]
```
正确生成独立对象数组（搭配 Array.from）：
```js
const arr = Array.from({length:2}, ()=>({num:1}));
arr[0].num = 99; // 互不影响
```

# 核心总结对照表
| API             | 核心用途                          | 关键点                            |
| --------------- | --------------------------------- | --------------------------------- |
| `...`扩展运算符 | 展开数组、浅拷贝、合并传参        | 浅拷贝，仅支持可迭代对象          |
| 数组空位        | ES5跳过，ES6统一转undefined       | 避免 `Array(n)` 创建空位数组      |
| `Array.from`    | 类数组/可迭代转数组，自带map      | 兼容NodeList、arguments，处理空位 |
| `Array.of`      | 替代 Array 构造函数，统一创建数组 | 单数字参数不会生成空位数组        |
| `arr.fill()`    | 批量填充数组，修改原数组          | 引用值填充共享内存，慎用对象      |
