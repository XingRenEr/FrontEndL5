# JavaScript 字符串完整知识点
## 一、创建字符串
### 1. 字面量（推荐）
```js
let s1 = '单引号';
let s2 = "双引号";
let s3 = `模板字符串`; // ES6，支持换行、插值
```
### 2. String 构造函数
```js
let str = new String('abc'); // 字符串对象，一般不用
typeof str; // "object"
typeof 'abc'; // "string" 原始字符串
```
### 模板字符串 `` ` ``
```js
let name = '小明';
let msg = `你好，${name}
支持换行`;
```

## 二、基础属性与取值
1. `.length` 获取字符长度
```js
'hello'.length; // 5
```
2. 通过下标读取字符（只读，不能修改）
```js
let s = 'abcd';
s[0]; // 'a'
s[0] = 'x'; // 无效，字符串不可变
```
3. `charAt(index)` 获取指定下标字符
```js
'abc'.charAt(1); // 'b'
'abc'.charAt(99); // '' 空字符串
```
4. `charCodeAt(index)` 获取字符 Unicode 编码
```js
'a'.charCodeAt(0); // 97
```

## 三、字符串查找方法（不改变原字符串）
### 1. indexOf / lastIndexOf
```js
let str = 'abcb';
str.indexOf('b'); // 1 首次出现下标
str.indexOf('x'); // -1 不存在返回-1
str.lastIndexOf('b'); // 3 从后往前找
// 第二个参数：起始查找位置
str.indexOf('b', 2); // 3
```
### 2. includes() 判断是否包含，返回布尔
```js
'hello'.includes('el'); // true
'hello'.includes('a'); // false
// 第二个参数：起始位置
'hello'.includes('e', 2); // false
```
### 3. startsWith / endsWith
```js
'abc123'.startsWith('abc'); // true
'abc123'.endsWith('123'); // true
```

## 四、截取字符串（不修改原串）
1. `slice(start, end)` 推荐，支持负数
```js
let s = '012345';
s.slice(1,3); // '12' 左闭右开
s.slice(2); // 从2截到末尾 '2345'
s.slice(-2); // 最后两位 '45'
s.slice(1,-1); // 1到倒数第二位
```
2. `substring(start, end)` 不支持负数，负数自动转为0
```js
s.substring(1,3); // '12'
s.substring(-2,3); // 等价 substring(0,3)
```
3. `substr(start, length)` 已废弃不推荐
```js
s.substr(1,2); // 从下标1截取2个字符 '12'
```

## 五、大小写转换
```js
'ABC'.toLowerCase(); // 'abc'
'abc'.toUpperCase(); // 'ABC'
```

## 六、去空格
```js
'  abc  '.trim(); // 'abc' 首尾空格
'  abc  '.trimStart(); // 只去开头
'  abc  '.trimEnd(); // 只去结尾
```

## 七、替换 replace
1. 基础：只替换第一个匹配
```js
let s = 'aa bb';
s.replace('a', 'x'); // 'xa bb'
```
2. 全局替换用正则 `/g`
```js
s.replace(/a/g, 'x'); // 'xx bb'
```
3. 第二个参数可以是回调函数
```js
'123'.replace(/\d/g, v => Number(v)*2); // '246'
```

## 八、分割、拼接、重复
### 1. split(分隔符) 字符串转数组
```js
'1,2,3'.split(','); // ['1','2','3']
'abc'.split(''); // ['a','b','c']
'hello'.split(); // 整个字符串作为一个元素数组
```
### 2. concat() 拼接（少用，+更简单）
```js
'a'.concat('b', 'c'); // 'abc'
'a' + 'b' + 'c'; // 同上
```
### 3. repeat(n) 重复字符串 ES6
```js
'6'.repeat(3); // '666'
```

## 九、填充补齐 padStart / padEnd
常用于补零（时间、编号）
```js
'5'.padStart(3, '0'); // '005' 总长度3，前面补0
'5'.padEnd(3, '0'); // '500' 后面补0
```

## 十、字符串不可变特性
JS 字符串**一旦创建不能修改**，所有方法都会返回新字符串：
```js
let str = 'abc';
str[0] = 'x';
console.log(str); // 'abc' 不变
let newStr = str.replace('a','x'); // 生成新字符串
```

## 十一、字符串与数组互转
```js
// 字符串转数组
let arr = 'hello'.split('');
// 数组转字符串
let str = arr.join('');
```

## 十二、常用小案例
1. 判断是否为空串
```js
str === ''
```
2. 反转字符串
```js
let res = 'abcd'.split('').reverse().join(''); // 'dcba'
```
3. 截取文件后缀
```js
let file = 'test.png';
let suffix = file.slice(file.lastIndexOf('.')); // '.png'
```
