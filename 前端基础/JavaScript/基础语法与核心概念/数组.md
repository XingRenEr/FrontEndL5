# JavaScript 数组完整教程
## 一、数组基础
### 1. 创建数组
```js
// 字面量（推荐）
const arr1 = [1, 2, "abc", true, null];
// 构造函数
const arr2 = new Array(10); // 长度10空数组
const arr3 = new Array(1, 2, 3); // [1,2,3]
// 空数组
const arr = [];
```

### 2. 读取/修改元素
数组下标从 `0` 开始：
```js
const arr = ["苹果", "香蕉"];
console.log(arr[0]); // 苹果
arr[1] = "橙子";
arr[10] = "葡萄"; // 中间空位为 empty
console.log(arr.length); // 11
```
- `length`：数组长度，可手动修改截断数组

### 3. 检测是否为数组
```js
Array.isArray(arr); // true
arr instanceof Array; // true
```

## 二、常用数组方法（分增删、遍历、转换、查找、过滤）
### （一）头尾增删（改变原数组）
1. `push()` 末尾添加，返回新长度
```js
const a = [1,2];
a.push(3,4); // [1,2,3,4]
```
2. `pop()` 删除最后一个，返回删除元素
```js
a.pop(); // 4，数组变为 [1,2,3]
```
3. `unshift()` 头部添加
```js
a.unshift(-1,0); // [-1,0,1,2,3]
```
4. `shift()` 删除第一个
```js
a.shift(); // 0，数组 [1,2,3]
```

### （二）任意位置增删改 splice（改变原数组）
语法：`arr.splice(起始索引, 删除个数, 新增元素...)`
```js
const arr = [10,20,30,40];
arr.splice(1,1); // 删除下标1，[10,30,40]
arr.splice(1,0,25); // 下标1插入25，[10,25,30,40]
arr.splice(2,1,99); // 下标2替换为99
```
又返回值，返回删掉的那一段数组

### （三）截取 slice（不改变原数组）
`slice(开始, 结束)` 左闭右开，结束省略截取到末尾
```js
const arr = [1,2,3,4];
arr.slice(1,3); // [2,3]
arr.slice(-2); // [3,4] 倒数两位
```

### （四）合并 concat（不改变原数组）
```js
[1,2].concat([3,4], 5); // [1,2,3,4,5]
```

### （五）翻转 reverse / 排序 sort（改变原数组）
```js
// 翻转
[1,2,3].reverse(); // [3,2,1]

// sort 默认按字符串排序
const num = [10,2,22];
num.sort(); // [10,2,22] 错误数字排序
// 数字升序
num.sort((a,b) => a - b);
// 降序
num.sort((a,b) => b - a);
```

## 三、数组遍历方法（重点）
### 1. forEach 单纯遍历，无返回值
```js
const arr = [1,2,3];
arr.forEach((item, index) => {
  console.log(index, item);
});
```

### 2. map 遍历并返回新数组（映射）
```js
const res = arr.map(item => item * 2);
// [2,4,6]
```

### 3. filter 过滤，返回符合条件新数组
```js
const res = arr.filter(item => item > 1);
// [2,3]
```

### 4. reduce 累加/聚合
语法：`reduce((累加器, 当前项) => {}, 初始值)`
```js
// 求和
const sum = arr.reduce((total, cur) => total + cur, 0); // 6
// 求最大值
const max = arr.reduce((t, c) => t > c ? t : c, 0);
```

### 5. some / every
- `some`：任意一项满足条件返回 `true`
- `every`：全部满足才返回 `true`
```js
arr.some(x => x > 2); // true
arr.every(x => x > 0); // true
```

## 四、查找元素
1. `indexOf(item)` 返回下标，不存在 `-1`
2. `lastIndexOf()` 从后往前找
```js
[1,2,3,2].indexOf(2); // 1
```
3. `find()` 返回第一个匹配元素
```js
[1,2,3].find(x => x > 2); // 3
```
4. `findIndex()` 返回第一个匹配下标
```js
[1,2,3].findIndex(x => x > 2); // 2
```
5. `includes()` 判断是否包含，返回布尔
```js
[1,2,3].includes(2); // true
```

## 五、数组与字符串互转
```js
// 数组转字符串 join
["a","b"].join("-"); // "a-b"

// 字符串转数组 split
"1,2,3".split(","); // ["1","2","3"]
```

## 六、ES6 数组拓展
### 1. 展开运算符 ...
```js
const a = [1,2];
const b = [...a, 3,4]; // [1,2,3,4]
const copy = [...a]; // 浅拷贝数组
```

### 2. Array.from
类数组（DOM集合、arguments）转数组
```js
Array.from("hello"); // ["h","e","l","l","o"]
Array.from({length:5}, (_,i)=>i) // [0,1,2,3,4]
```

### 3. fill 填充数组
```js
new Array(3).fill(0); // [0,0,0]
```

### 4. flat / flatMap 扁平化
```js
[1,[2,[3]]].flat(2); // [1,2,3] 深度2扁平化

// 无限扁平化（常用）
const arr = [1, [[[[2]]]], 3];
console.log(arr.flat(Infinity)); // [1,2,3]
```

flatMap ≡ 先执行 map 遍历映射 → 再自动执行 flat(1)
固定只能扁平化1层，无法自定义深度，这是和 flat 最大区别。

## 七、数组去重常用写法
```js
// Set 最简
const unique = [...new Set([1,2,2,3])]; // [1,2,3]

// filter + indexOf
const arr = [1,2,2];
arr.filter((v,i) => arr.indexOf(v) === i);
```

## 八、浅拷贝 vs 深拷贝
- 浅拷贝：`[...arr]` / `slice()` / `concat()`，只能拷贝一层，嵌套对象仍共享引用
- 深拷贝：复杂嵌套用 `JSON.parse(JSON.stringify(arr))` 或递归
