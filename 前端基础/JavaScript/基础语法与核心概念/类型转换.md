# JavaScript 类型转换完整详解
JS 类型转换分两大类：**隐式转换（自动）**、**显式转换（手动强制）**
原始类型：`Number`、`String`、`Boolean`、`Null`、`Undefined`、`Symbol`、`BigInt`
引用类型：`Object`（数组、函数、普通对象等）

## 一、显式强制转换（手动可控）
### 1. 转数字 Number()
规则：
1. 纯数字字符串 → 对应数字
2. 空串/空格串 → `0`
3. 非纯数字 → `NaN`
4. `null → 0`；`undefined → NaN`
5. `true → 1`；`false → 0`
6. 对象：先调用 `valueOf()` → 再 `toString()`，最后转数字
```js
Number('123')      // 123
Number('')         // 0
Number('abc')      // NaN
Number(null)       // 0
Number(undefined)  // NaN
Number(true)       // 1
Number([])         // 0
Number(['123'])    // 123
Number({})         // NaN
```
简化函数：`parseInt()` / `parseFloat()`（只提取开头数字）
```js
parseInt('12abc') // 12
parseFloat('3.14px') // 3.14
```

### 2. 转字符串 String()
所有类型直接转为字面文本，无歧义
```js
String(123)       // "123"
String(null)      // "null"
String(undefined) // "undefined"
String(true)      // "true"
String([])        // ""
String([1,2])     // "1,2"
String({})        // "[object Object]"
```
简写：任意值 + `''`

### 3. 转布尔 Boolean()
**假值（6个，转后为 false）**：
`0`、`NaN`、`''`、`null`、`undefined`、`false`
其余所有值均为 `true`（空数组、空对象也是 true）
```js
Boolean(0)        // false
Boolean('')       // false
Boolean(null)     // false
Boolean([])       // true
Boolean({})       // true
Boolean(-1)       // true
```
简写：`!!值` 双重取反强制转布尔

## 二、隐式自动转换（运算时自动触发）
### 1. 字符串拼接：`+` 一侧有字符串，全部转字符串
```js
1 + '2'    // "12"
null + 'a' // "nulla"
[] + {}    // "[object Object]"
```

### 2. 数学运算：`- * / %` 全部转数字
```js
'5' - 2    // 3
'5' * '2'  // 10
'abc' - 1  // NaN
null - 1   // -1
```

### 3. 一元加号 `+x`：快速转数字
```js
+'123'  // 123
+''     // 0
+true   // 1
```

### 4. 逻辑运算符 `|| && !` 转布尔
```js
!0        // true
!''       // true
![]       // false
```

### 5. 相等判断 ==（宽松相等，自动转换）
严格相等 `===` 不转换，类型不同直接 false
宽松相等转换规则：
1. `null == undefined → true`
2. 数字与字符串：字符串转数字
3. 布尔与任意：布尔转数字
4. 对象和原始：对象调用 `valueOf/toString`
```js
'' == 0    // true
'1' == true// true
null == undefined // true
[] == ''   // true
[] == 0    // true
{} == '[object Object]' // true
```

## 三、对象转换底层机制（ToPrimitive）
对象参与运算时，JS 会调用内部 `ToPrimitive(值, 期望类型)`
1. 期望数字（减法、比较、Number()）：
   先执行 `obj.valueOf()`，结果非对象则使用；否则执行 `obj.toString()`
2. 期望字符串（拼接、String()）：
   先执行 `obj.toString()`，再 `valueOf()`
```js
const obj = {
  valueOf(){ return 10 },
  toString(){ return "20" }
}
obj + 1  // 11（优先valueOf）
'' + obj // "20"（优先toString）
```

## 四、高频经典转换例题
```js
[] + {}   // "[object Object]"
{} + []   // 0（{}被识别为空代码块，只剩 +[] → 0）
[] == ![] // true 拆解：![]=false → []==0 → ""==0 → true
1 + 2 + '3' // "33"
'1' + 2 + 3 // "123"
```

## 五、转换速记总结
1. 转数字：`Number(x)` / `+x` / `x - 0`
2. 转字符串：`String(x)` / `x + ''`
3. 转布尔：`Boolean(x)` / `!!x`
4. `+` 两边有字符串 → 拼接；`-*/` 一律转数字
5. 宽松 `==` 自动类型转换，开发推荐永远用 `===` 严格相等
6. 空数组、空对象、负数、非空字符串都是真值
