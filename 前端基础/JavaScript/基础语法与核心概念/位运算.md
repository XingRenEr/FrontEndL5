# JS 位运算详解
位运算直接操作数字的**二进制补码**，所有运算前会把数值转为 32 位有符号整数（超出范围自动截断）。

## 一、7 种位运算符汇总
| 运算符 | 名称       | 作用                     |
| ------ | ---------- | ------------------------ |
| `&`    | 按位与     | 两位都为1，结果才是1     |
| `|`    | 按位或     | 任意一位为1，结果为1     |
| `^`    | 按位异或   | 两位不同为1，相同为0     |
| `~`    | 按位取反   | 0变1，1变0（一元运算符） |
| `<<`   | 左移       | 二进制整体左移，右侧补0  |
| `>>`   | 有符号右移 | 右侧丢弃，左侧补符号位   |
| `>>>`  | 无符号右移 | 右侧丢弃，左侧统一补0    |

## 1. 按位与 `&`
规则：同1才1，否则0
```js
// 5 二进制  0101
// 3 二进制  0011
console.log(5 & 3); // 0001 → 1
```
常用场景：判断奇偶 `n & 1`，结果1为奇数，0偶数。
```js
6 & 1 // 0 偶数
7 & 1 // 1 奇数
```

## 2. 按位或 `|`
规则：有1则1，全0才0
```js
// 5:0101  3:0011
console.log(5 | 3); // 0111 → 7
```
常用：小数取整 `num | 0`
```js
3.9 | 0  // 3
-2.6 | 0 // -2
```

## 3. 按位异或 `^`
规则：相同0，不同1
```js
// 5:0101 3:0011
console.log(5 ^ 3); // 0110 → 6
```
实用技巧：
1. 交换两个变量，不借助中间值
```js
let a = 2, b = 5;
a = a ^ b;
b = a ^ b;
a = a ^ b;
// a=5, b=2
```
2. 判断两数是否相等 `a ^ b === 0`

## 4. 按位取反 `~`
公式：`~x = -(x + 1)`
```js
~5   // -6
~0   // -1
~-3  // 2
```
搭配 `~~` 快速取整（等价 `Math.trunc`）
```js
~~4.8  // 4
~~-7.2 // -7
```

## 5. 左移 `<<`
整体左移，低位补0，等价 `n * 2^k`
```js
2 << 1 // 4  2*2
3 << 2 // 12 3*4
```

## 6. 有符号右移 `>>`
高位补**符号位**，等价向下取整除以 2 的幂
```js
8 >> 1  // 4
-8 >> 1 // -4
7 >> 2  // 1
```

## 7. 无符号右移 `>>>`
高位统一补0，负数运算会变成超大正数
```js
8 >>> 1  // 4
-1 >>> 0 // 4294967295（32位全1）
```

# 二、高频实用场景
1. 取整数：`~~num` / `num | 0`
2. 判断奇偶：`n & 1`
3. 乘2/除2快速运算：`<<` / `>>`
4. 不用临时变量交换数字：`^`
5. 权限掩码组合（多状态合并判断）
```js
const READ = 1 << 0; // 001
const WRITE = 1 <<1; // 010
const DEL = 1<<2;    // 100
let perm = READ | WRITE;
// 判断是否有读权限
if(perm & READ){}
```

# 三、注意事项
1. JS数字是64位浮点数，位运算会强制转为32位有符号整数，大于 2³¹-1 会失真
2. 负数二进制存储是**补码**，取反、右移结果容易和直觉不符
3. 位运算优先级低于加减乘除，混合运算建议加括号
