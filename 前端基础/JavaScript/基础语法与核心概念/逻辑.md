# JavaScript 逻辑运算完整教程
JS 逻辑运算分三类：**逻辑与 &&、逻辑或 ||、逻辑非 !**，同时存在**短路特性**、**真值/假值**、**空值合并 ??**、**可选链 ?.** 扩展逻辑。

## 一、真值 & 假值（逻辑运算基础）
逻辑运算不会只返回 `true/false`，而是返回**原值**，判断规则靠真假值：
### 假值（共6个，其余全为真值）
`false`、`0`、`""`空字符串、`null`、`undefined`、`NaN`
### 真值
所有非假值：`1、-1、"abc"、[]、{}、Infinity` 等

## 二、基础逻辑运算符
### 1. 逻辑非 `!` 取反
规则：先转为布尔值，再取反；`!!` 可快速转布尔
```js
!0        // true
!"hello"  // false
!null     // true
!!123     // true（双重取反，等价Boolean()）
!!""      // false
```

### 2. 逻辑与 `a && b` 短路与
**短路规则**：
1. 如果 `a` 是**假值**，直接返回 `a`，不执行后面 b
2. 如果 `a` 是**真值**，返回 `b`
```js
0 && 100     // 0 （a假，直接返回a）
20 && "js"   // "js"（a真，返回b）
null && true // null
[] && {}     // {}
```
常用场景：判断存在再执行
```js
let user = { name: "小明" };
user && user.name && console.log(user.name);
```

### 3. 逻辑或 `a || b` 短路或
**短路规则**：
1. 如果 `a` 是**真值**，直接返回 `a`，不执行后面 b
2. 如果 `a` 是**假值**，返回 `b`
```js
10 || 20     // 10（a真，直接返回a）
"" || "默认" // "默认"（a假，返回b）
NaN || 0     // 0
```
常用场景：设置默认值
```js
function say(name) {
  name = name || "匿名";
  console.log(name);
}
say("张三"); // 张三
say();       // 匿名
```

## 三、运算符优先级
`!` > `&&` > `||`
```js
false || true && false
// 先算 true && false → false
// 再算 false || false → false
```

## 四、ES6 新增逻辑运算符（避坑替代 ||）
### 1. 空值合并运算符 `??`
仅当左侧为 `null / undefined` 才返回右侧；`0、""、false` 视为有效数据，不会走默认
解决 `||` 的缺陷：`0`、空字符串会被当成假值覆盖
```js
const num = 0;
console.log(num || 10);  // 10 （错误，0被覆盖）
console.log(num ?? 10);   // 0   （正确，0是有效值）

let data = null;
console.log(data ?? "空数据"); // "空数据"
```
限制：**不能和 &&、|| 混用**，必须加括号
```js
// 报错：null || undefined ?? "xxx"
(null || undefined) ?? "默认值" // 正确
```

### 2. 可选链操作符 `?.`
安全读取深层对象属性，避免 `Cannot read property 'xxx' of undefined`
```js
const info = null;
info?.user?.age; 
// 等价：info && info.user && info.user.age
```
搭配 ?? 实现安全取值+默认值
```js
info?.user?.name ?? "未知姓名"
```

## 五、逻辑运算经典示例
### 示例1：短路赋值
```js
let a = 5;
let b = a && a * 2; // 10
let c = "" || "test"; // test
```

### 示例2：if 判断简写
```js
// 传统
if (name) {}
// 简写，满足才执行
name && console.log("有名字");
```

### 示例3：真假值对比
```js
console.log(Boolean(""));     // false
console.log(Boolean([]));     // true 空数组是真值
console.log(Boolean({}));     // true 空对象是真值
```

## 六、常见坑
1. `||` 会把 `0、""、false` 当作空，优先用 `??` 做默认值
2. `&&` 和 `||` 返回**原值**，不一定是布尔值
3. `!!` 才是标准转布尔，不要直接用 `a && b` 判断布尔结果
4. `??` 只能判断 null/undefined，和 || 逻辑完全不同
