# JS 表单与表单API 完整知识点梳理
## 21. 表单 API（Form 内置接口）
### 一、获取表单元素
```js
// 1. 通过id
const form = document.getElementById('myForm')
// 2. forms集合（页面所有表单）
const forms = document.forms
const form2 = document.forms[0] // 下标
const form3 = document.forms.login // name属性
// 3. querySelector
const form4 = document.querySelector('form')
```

### 二、Form 表单内置属性（API）
| 属性            | 作用                                                       |
| --------------- | ---------------------------------------------------------- |
| `form.elements` | 获取表单内所有表单控件集合（input/select/textarea/button） |
| `form.length`   | 表单控件总数，等价于 `elements.length`                     |
| `form.name`     | 表单name属性                                               |
| `form.action`   | 表单提交地址                                               |
| `form.method`   | 提交方式 GET/POST                                          |
| `form.enctype`  | 编码格式（文件上传用 multipart/form-data）                 |
| `form.target`   | 提交打开窗口 _self/_blank                                  |

### 三、核心表单 API 方法
1. **submit() 手动提交表单**
```js
form.submit() // 不会触发submit事件
```
2. **reset() 重置表单**
```js
form.reset() // 清空输入，恢复默认值，触发reset事件
```

### 四、表单控件操作（elements）
```js
// 通过name获取控件
const username = form.elements.username
// 取值赋值
username.value = 'test'
console.log(username.value)

// 单选/复选框
form.elements.gender.value
form.elements.agree.checked // 布尔值
```

### 五、表单事件 API
1. `submit`：表单提交时触发（可阻止默认提交）
```js
form.addEventListener('submit', e => {
  e.preventDefault() // 阻止页面刷新提交
  // 表单校验逻辑
})
```
2. `reset`：重置表单触发
3. 控件通用事件：`input`（实时输入）、`change`（失去焦点/切换值）、`focus`、`blur`

### 六、FormData API（提交数据核心）
用于异步上传、序列化表单数据
```js
// 快速序列化整个表单
const fd = new FormData(form)
// 增删查改
fd.append('key', 'val')
fd.get('username')
fd.set('username', '123')
fd.delete('pwd')
// 遍历
for (const [k, v] of fd) {}

// axios/fetch上传
fetch('/api/login', {
  method: 'POST',
  body: fd
})
```

## 22. 基础表单操作（原生DOM操作表单控件）
### 1. 常用表单控件
`<input>`、`<textarea>`、`<select>`、`<option>`、`<button>`、`<label>`

### 2. input 通用取值
```html
<input type="text" name="user" id="user">
```
```js
const input = document.querySelector('#user')
// 获取输入内容
console.log(input.value)
// 设置默认值
input.value = '默认文字'

// 禁用/只读
input.disabled = true
input.readOnly = true

// 焦点控制
input.focus()
input.blur()
```

### 3. 单选框 radio
```html
<input type="radio" name="sex" value="man">男
<input type="radio" name="sex" value="woman">女
```
```js
const radios = document.querySelectorAll('input[name="sex"]')
radios.forEach(item => {
  if (item.checked) console.log(item.value)
})
```

### 4. 复选框 checkbox
```js
const check = document.querySelector('input[name="agree"]')
console.log(check.checked) // true/false
check.checked = true // 选中
```

### 5. 下拉选择 select
```html
<select name="city">
  <option value="gz">广州</option>
  <option value="sz">深圳</option>
</select>
```
```js
const sel = document.querySelector('select')
// 当前选中值
sel.value
// 选中下标
sel.selectedIndex
// 获取选中option
sel.options[sel.selectedIndex]
```

### 6. textarea 多行文本
```js
const textarea = document.querySelector('textarea')
textarea.value // 取值，innerHTML无效
```

### 7. 文件上传 input[type=file]
```js
const fileInput = document.querySelector('input[type=file]')
// 获取文件列表
const files = fileInput.files
const file = files[0] // 单个文件
```

### 8. 简单表单校验示例
```js
form.addEventListener('submit', e => {
  e.preventDefault()
  const uname = form.elements.username.value.trim()
  if (!uname) {
    alert('用户名不能为空')
    return
  }
  console.log('表单数据合法，发起请求')
})
```
